Overview

This cop audits variables in casks.

Examples:

# bad
cask do
  arch = Hardware::CPU.intel? ? "darwin" : "darwin-arm64"
end

# good
cask 'foo' do
  arch arm: "darwin-arm64", intel: "darwin"
end
